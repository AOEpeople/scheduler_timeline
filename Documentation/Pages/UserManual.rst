Users manual
============

All you need to do to enable this extension is to install it. There is no configuration needed, when installed you will have a new backend module in the backend under "Admin Tools".

.. image:: /Images/UserManual/BE_Module_Icon.png
.. image:: /Images/UserManual/BE_Module_Icon_v7.png

When you click "Scheduler Timeline" you will see a screen similar to:

.. image:: /Images/UserManual/BE_Module.png

You can see more information about the task when hovering the entry on the timeline.

.. image:: /Images/UserManual/TaskHover.png
